# Nahla’s Food Game (Snack Quest)

This project is a gamified behavior reinforcement web application designed to encourage consistent seated eating through an interactive story-based experience.

It was originally built to solve a real-world problem—helping a young child stay engaged and consistent during meals—by turning the experience into a reward-driven game instead of a routine task.

## Overview

Instead of relying on reminders or corrections, this system uses story progression and rewards to reinforce positive behavior.

The user becomes the “hero,” and progress in the story is directly tied to completing actions like taking bites and staying seated.

This creates a feedback loop where engagement drives behavior.

---

## Key Features

- Story-based progression system  
- Bite tracking with real-time feedback  
- Reward and power-up system  
- Seat-streak monitoring  
- Text-to-speech for immersive storytelling  
- Hidden parent mode for customization  
- Local storage for persistent progress
